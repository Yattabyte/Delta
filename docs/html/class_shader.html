Detailed Description

An entire OpenGL vertex/fragment shader program.

An encapsulation of an OpenGL vertex & fragment shader program. Responsible for loading the files associated with this program from disk, and forming the program. Also provides support for explicitly setting uniform values for a given attribute location. Supports binary representation.
