Detailed Description

A collection of textures which together form 1 or more PBR surfaces.

A material is a collection of textures that are used to approximate a real surface, such as:

  • Albedo
  • normal
  • metalness
  • roughness
  • height (WIP)
  • occlusion
    Note
    supports omission of any or all of the files.
    expects all textures in a material to be the same dimension, and will forcefully resize them (in memory).
    can contain multiple sets of these textures, where each set of 6 form 1 skin for an object.
    owns many Shared_Image objects.
