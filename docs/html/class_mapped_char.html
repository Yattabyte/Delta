Detailed Description

template<typename T>
class MappedChar< T >

A std::map that uses a constant char array as the keys, and stores values of type <T>.
Uses the adapter pattern to hide some of the complexity and reduce the amount of redundant code written. Also provides easier insertion, lookup, and boolean find operations.

Parameters
<T>any type to store.
